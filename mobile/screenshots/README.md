# App Screenshots

This directory contains screenshots of the Expense Tracker Mobile App.

## Screenshots Included

1. **dashboard.png** - Main dashboard with budget progress, expense summary, and charts
2. **add-expense.png** - Add expense form with date picker and category selection
3. **history.png** - Expense history with search and filtering
4. **edit-expense.png** - Edit expense form with pre-filled data
5. **budget-management.png** - Budget management screen with add/edit functionality

## Screenshot Descriptions

### Dashboard Screen
- Budget progress indicator (54% used)
- Expense summary (₹26,890.21 actual vs ₹50,000.00 budget)
- Total statistics (5 transactions, 3 categories)
- Time filters (DAILY, WEEKLY, MONTHLY, YEAR)
- Expense distribution pie chart
- Monthly expenses vs budget line chart

### Add Expense Screen
- Date picker with calendar interface
- Category dropdown (Needs/Wants)
- Amount input with ₹ currency symbol
- Optional comments field
- Form validation and error handling

### History Screen
- Search functionality across all fields
- Advanced filtering (category, time period)
- Expense list with edit/delete actions
- Total expense summary
- Action buttons for each expense

### Edit Expense Screen
- Pre-filled form with existing data
- Same interface as Add Expense
- Update functionality with validation
- Navigation back to History

### Budget Management Screen
- Add budget form (month/year/amount)
- Budget list with existing budgets
- Edit and delete budget options
- Visual feedback and progress indicators
